Child Mind Institute Problematic Internet Use - Kaggle Competition Silver Medal

Achieved top 3% ranking in predicting problematic internet usage patterns using ensemble learning with LightGBM, XGBoost and CatBoost models. Implemented efficient data preprocessing with Polars and KNNImputer for missing values handling.

Secure Multi-Layered File System with Comprehensive Access Control

Firebase Emulation with Citi Bike Station Query Platform

Aspect-based Customer Review Analysis

Developed an aspect-based sentiment analysis classifier with PyTorch, tailored for restaurants and laptops, targeting the SemEval 2014 Task 4 dataset. By discerning topics within customer reviews, classifier accomplished 95.6% accuracy.
